# ABRIR\_SIGPAC

Abre una ventana de Sig Pac en las coordenadas donde está el cursor en la ventana de dibujo.

## Parámetros

No admite parámetros.

## Características de la orden

| Tipo de orden | [Orden inmediata](abrir-sigpac.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | Zooms/Abrir una ventana de Sigpac en la posición actual |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión |  |
| Variables relacionadas | No tiene variables relacionadas |
| Nombre interno | {481ACFE7-8826-486E-BB15-5E495F3D4D71} |
