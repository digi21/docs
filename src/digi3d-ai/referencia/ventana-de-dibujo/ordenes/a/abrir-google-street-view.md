# ABRIR\_GOOGLE\_STREET\_VIEW

Abre una ventana de Google Street View en las coordenadas donde está el cursor en la ventana de dibujo.

## Parámetros

No admite parámetros.

## Características de la orden

| Tipo de orden | [Orden inmediata](abrir-google-street-view.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | Zooms/Abrir una ventana de Google Street View en la posición actual... |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión |  |
| Variables relacionadas | No tiene variables relacionadas |
| Nombre interno | {C84757D8-EB3E-4ED1-918B-86AB14F40899} |
