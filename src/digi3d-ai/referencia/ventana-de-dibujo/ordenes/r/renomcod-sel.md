# RENOMCOD\_SEL

Cambia el código correspondiente otro código a las entidades seleccionadas.

## Parámetros

| Número de parámetro | Descripción | Opcional |
| :--- | :--- | :--- |
| 1 | Código origen | No |
| 2 | Código destino | No |

## Observaciones

No se han registrado observaciones para esta orden.
