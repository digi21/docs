# PARAMETROS\_CAMARA\_CONICA

Asigna parámetros de la cámara cónica.

## Parámetros

| Número de parámetro | Descripción | Opcional |
| :--- | :--- | :--- |
| 1 | Campo de visión (Fov) | No |
| 2 | Distancia cercana | No |
| 3 | Distancia lejana | No |
