# PROYECTA\_ENTRE\_CURVAS

Genera una triangulación virtual con las líneas seleccionadas y proyecta los puntos situados dentro del límite sobre la triangulación virtual.

## Parámetros

No admite parámetros.

## Observaciones

No se han registrado observaciones para esta orden.
