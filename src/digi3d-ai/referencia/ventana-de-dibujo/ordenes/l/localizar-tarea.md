# LOCALIZAR\_TAREA

Esta orden solicita al usuario que digitalize un punto y se localiza y selecciona la tarea que hubiera en ese punto (si es que había alguna).

## Parámetros

No admite parámetros.

## Características de la orden

| Tipo de orden | [Orden interactiva](localizar-tarea.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | Ventana/Tareas/Localizar tarea |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión | DigiNG.OrdenesStandard.dll |
| Variables relacionadas | No tiene variables relacionadas |
| Nombre interno | {C3CC6DE5-A638-447D-B68C-8AA3ECD69479} |
