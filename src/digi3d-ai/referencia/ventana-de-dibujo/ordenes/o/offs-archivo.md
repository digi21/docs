# OFFS\_ARCHIVO

Desactiva códigos en la ventana fotogramétrica para un determinado número de archivo.

## Parámetros

| Número de parámetro | Descripción | Opcional |
| :--- | :--- | :--- |
| 1 | Archivo | No |
| 2 | Código o códigos | Si |

## Observaciones

No se han registrado observaciones para esta orden.
