# UNIR\_LINEAS\_VISIBLES

Une las lineas visibles en pantalla (siempre que tengan el mismo código y continuidad geométrica).

## Parámetros

No admite parámetros.

## Observaciones

No se han registrado observaciones para esta orden.

| Tipo de orden | [Orden inmediata](unir-lineas-visibles.md) |

| Repite automáticamente | No |
| Opción del menú donde aparece la orden | Análisis geométricos/Unir líneas/Independientemente del número de líneas que lleguen al nodo/Líneas visibles |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
