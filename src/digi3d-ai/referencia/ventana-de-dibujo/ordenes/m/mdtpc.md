# MDTPC

Genera una nube de puntos automáticamente por correlación.

## Parámetros

| Número de parámetro | Descripción | Opcional |
| :--- | :--- | :--- |
| 1 | Código | No |
| 2 | Geometría del límite | No |
| 3 | Z mínima | No |
| 4 | Z máxima | No |
| 5 | Distancia | No |
| 6 | Distancia máxima de desplazamiento en Z | No |
| 7 | Semiancho 8x | No |
| 8 | Semiancho 4x | No |
| 9 | Semiancho 1x | No |
| 10 | Factor mínimo de correlación permitido | No |
| 11 | Oportunidades para buscar el factor de correlación mínimo | No |
| 12 | Píxeles de incremento de la espiral | No |
| 13 | Borrar los puntos que no cumplan la condición (0/1) | No |
| 14 | Utilizar muestras de entrenamiento (0/1) | No |
| 15 | Archivo de muestras | Si |
| 16 | Tamaño de muestra en X | Si |
| 17 | Tamaño de muestra en Y | Si |
