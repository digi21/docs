# ESCALA\_DIBUJO

Indica la escala con la que se representa el patrón en pantalla.

## Parámetros

| Número de parámetro | Descripción | Valores | Opcional |
| :--- | :--- | :--- | :--- |
| 1 | Valor numérico | Número real | Si |

### Ejemplos

`ESCALA_DIBUJO=1000`

Asigna como escala de dibujo el valor 1000

`ESCALA_DIBUJO=?`

Muestra el valor actual de la escala de dibujo

## Características de la orden

| Tipo de orden | [Variable real](escala-dibujo.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | _Esta orden no tiene asociada ninguna opción de menú_ |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión | DigiNG.OrdenesStandard.dll |
| Variables relacionadas | No tiene variables relacionadas |
| Nombre interno | {E992D308-2F0B-4f44-B7D7-09999B9D42BE} |
