# IR\_TENTATIVO

Indica si, al tentativar \(enganchar el cursor\) sobre una entidad, el restituidor se desplaza a las coordenadas tentativadas.

## Parámetros

Ver [parámetros de variables booleanas](/digi3d-ai/referencia/ventana-de-dibujo/variables/variables-booleanas.md).

## Observaciones

Cuando esta variable está activa, al realizar un tentativo sobre una entidad el restituidor se mueve automáticamente a la posición enganchada. Si se desactiva, el tentativo calcula las coordenadas pero el restituidor no se desplaza a ellas.

Esta variable está activada de forma predeterminada.
