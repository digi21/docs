# Digi21.DigiNG.Plugin.QualityControl

Proporciona tipos para implementar controles de calidad en la ventana de dibujo de Digi3D.AI.

## Clases

|  |  |
| :--- | :--- |
| [DatabaseAttributeException](/digi3d-ai/programacion/.net/referencia/digi21.diging.plugin/digi21.diging.plugin.qualitycontrol/excepciones/databaseattributeexception.md) | Esta clase define una excepción para informarle a Digi3D.AI que un atributo de base de datos no cumple con el control de calidad. |
| [DigitizeAsPointException](/digi3d-ai/programacion/.net/referencia/digi21.diging.plugin/digi21.diging.plugin.qualitycontrol/excepciones/digitizeaspointexception.md) | Esta clase define una excepción para informarle a Digi3D.AI que una determinada geometría debería digitalizarse como punto. |
| [GeometricRelationException](/digi3d-ai/programacion/.net/referencia/digi21.diging.plugin/digi21.diging.plugin.qualitycontrol/excepciones/geometricrelationexception.md) | Esta clase define una excepción para informarle a Digi3D.AI que se ha localizado un error de control de calidad entre dos geometrías. |
| [GeometryException](/digi3d-ai/programacion/.net/referencia/digi21.diging.plugin/digi21.diging.plugin.qualitycontrol/excepciones/geometryexception.md) | Esta clase define una excepción para informarle a Digi3D.AI que una geometría no cumple con los criterios de control de calidad. |

## Interfaces

|  |  |
| :--- | :--- |
| [ITrigger](/digi3d-ai/programacion/.net/referencia/digi21.diging.plugin/digi21.diging.plugin.qualitycontrol/interfaces/itrigger/) | Este interfaz define los métodos que debe implementar una clase que realice control de calidad. |
