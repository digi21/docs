# Licenciamiento

En esta sección de la configuración se configuran los parámetros de la licencia del producto.

| Parámetro | Descripción                                                                                                                                                                                                                         |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Key ID    | Introduce aquí el Key ID de la llave de protección de Digi21 a utilizar.; Puedes averiguar el Key ID entrando en [http://localhost:1947/_int_/devices.html](http://localhost:1947/_int_/devices.html)|
