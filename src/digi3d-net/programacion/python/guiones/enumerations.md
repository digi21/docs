# Enumeraciones

Módulo: `digi3d`

## InputDevice

Dispositivo de entrada de la ventana de dibujo (propiedad
[`DrawingView.input_device`](drawing-view.md#otras-opciones)).

| Valor | Descripción |
|---|---|
| `Mouse` | El ratón. |
| `PhotogrammetricWindow` | La ventana fotogramétrica. |

## MusicType

Sonidos estándar de Digi3D.NET (función [`music`](functions.md#mensajes-y-avisos)).

| Valor | Descripción |
|---|---|
| `Beep` | Pitido. |
| `End` | Fin de orden. |
| `Error` | Error. |
| `In` | Entrar. |
| `Out` | Salir. |

## BallonType

Icono del globo informativo (función [`show_ballon`](functions.md#mensajes-y-avisos)).

| Valor | Descripción |
|---|---|
| `None` | Sin icono. |
| `Info` | Información. |
| `Warning` | Aviso. |
| `Error` | Error. |

## TaskType

Tipo de una [tarea](tasks.md).

| Valor | Descripción |
|---|---|
| `Info` | Mensaje informativo. |
| `Warning` | Aviso. |
| `Error` | Error. |

## FillType

Tipo de relleno de una [Representation](representation.md).

| Valor | Descripción |
|---|---|
| `None` | Sin relleno. |
| `Color` | Relleno con color. |
| `Bitmap` | Relleno con bitmap. |
